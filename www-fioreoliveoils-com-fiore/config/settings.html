Put your additional text/images/videos into pages and create a link list that contains links to each of these pages. Then select the linklist you've made below.
* By default, these areas have no border. To add a border, enter the text 'textwithborder' or 'imagewithborder' into the name of the link list item name.
* To show as one column width, make sure you have the word 'small' in the name of your link list item
(eg. 'Homepage video imagewithborder small').

Fonts

To use custom Google Web Fonts, go to google.com/webfonts and pick your font.
